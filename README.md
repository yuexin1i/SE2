# 💰 Expense Tracker (記帳視覺化工具)

>  Python 開發的現代化雙視窗記帳系統，提供流暢的使用者體驗與資料視覺化。

## 🛠️ 安裝與執行 (Installation)

### 前置需求
本專案依賴 Python 3.x 以及以下第三方套件：
```bash
pip install tkcalendar matplotlib
```
### 啟動程式
請直接執行主程式 `main.py`
